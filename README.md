## 常见问题

### 什么是 Claude Code 技能？

Claude Code 技能是扩展 Claude Code 功能的可复用能力。技能可通过斜杠命令调用，为视频生成、社媒管理等特定任务提供专业领域知识和工具集成。

### 什么是 MCP 服务器？

MCP（Model Context Protocol）服务器提供 AI 助手可调用的工具接口。在自媒体工作流中，MCP 服务器可以连接社交媒体 API、视频编辑工具和内容管理系统。
